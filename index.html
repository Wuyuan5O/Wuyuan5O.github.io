Mybatis之#{}和 ${}

发表于 2021-05-30 更新于 2021-06-05 分类于 Mybatis

#{} 是预编译处理，${}是字符串替换。

  • Mybatis 在处理 #{}时，会将 sql 中的 #{}替换为?，调用 PreparedStatement 的set 方法来赋值

  • Mybatis 在处理${}时，就是把${}替换成变量的值

所以使用 #{}可以有效的防止 SQL 注入，提高系统安全性 。

ps：mybatis 执行的本质还是 SQL，因此回归本质可以简单理解为一个对于 PreparedStatement ，一个对应 Statement。
