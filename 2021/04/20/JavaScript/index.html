2.1 分号

js语句可以不用分号结尾，但是在编译的时候这些没有用分号结尾的语句会被放在一行。为了避免出错一般都用分号结尾。

2.3 声明变量

格式：变量声明类型 变量名;

2.3.1 变量声明类型

  • var
  1. var只有函数作用域，没有块作用域，可以声明全局/局部变量（在一个函数内声明的变量，只在该函数有效）
  2. var定义的变量不能跨函数访问，但是可以跨块访问
  3. var 定义的变量如果不初始化会输出undefined，但不会报错
  4. 可以重复定义，后定义的会覆盖先定义的
  • const
  1. const 定义的变量不可以修改，并且必须进行初始化
  2. const一般用来声明常量，并且声明的常量不允许改变的，(只读属性)，因此必须在声明的同时进行赋值
  3. const也是块级作用域，与let相同，只能在块作用域里访问，存在暂时性死区，不存在变量声明提前，不允许重复定义
  • let
  1. let是块级作用域，函数内部使用let定义后，对函数外部无影响
  2. let定义的变量只能在块作用域中访问，不能跨块访问，更不能跨函数访问
  3. 不能变量声明提前，否则会报错
  4. 不能重复定义，否则会报错

因为JavaScript是一门弱类型语言，所以无法确定变量保存具体类型的值，并且变量的类型也可以随时发生变换。

class继承实质上是原型继承。

原型继承中，每个对象的__proto__指向自己的父类，并且最终都会指向Object，

9.2 DOM

Document Objec Model，文档操作模型。

HTML页面可以看做一棵树，之中每个元素都能看做是他的节点。

操作DOM要先获得要操作的节点：

document.getElementById("div1")

对这些节点可以：

  • 更新
  • 遍历
  • 增加
  • 删除

更新

  1. innerText(节点中新的内容) 纯文本
  2. innerHTML(节点中新的内容) 内容带有html标签的话可以被解析

两个方法都会覆盖节点中的内容。

删除

怎么删除？

通过父节点来删除子节点。

要注意的是，删除的过程是动态的，一个节点的children数组在进行删除操作的时候里面的元素是动态变化的。

增加

创建一个节点

document.createElement(节点标签) 返回一个节点元素

新增节点

appendChild(要加进来的节点变量) 在子节点序列的最后加一个

如果节点内容为空，也可以使用更新那里的那两个方法。

10. 表单的操作

表单绑定提交事件

通过为表单的onsubmit绑定函数，当点击提交时会在函数中进行相关的数据处理和验证。

关于点击了submit按钮之后：

如果onsubmit有绑定函数，则会先进入函数再进行表单是否提交的判定。

关于onsubmit中的return：

  1. 如果onsubmit的值是简单的true，false或者函数则点击submit按钮后并不会阻止提交，并且会跳转。
  2. 如果是return true则验证通过，提交信息并跳转；return false提交会被阻止，并且不会跳转。是true还是false可以看对应绑定的函数的返回值。
