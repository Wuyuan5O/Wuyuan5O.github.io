双指针

发表于 2021-07-31 分类于 LeetCode ， 刷题记录

双指针又分为快慢指针和对撞指针。

快慢指针

将快指针定义为fast，慢指针定义为slow。两个指针在同一侧以不同的策略开始扫描数组，直到符合指定条件后停止。

习题：

  • 88.Merge Sorted Array (Easy)
  • 141.Linked List Cycle (Easy)
  • 524.Longest Word in Dictionary through Deleting (Medium)
  • 75.Sort Colors (Medium)

对撞指针

将左指针定义为left，右指针定义为right。两个指针分别在左右两侧往中间扫描数组，直到left>=right或者符合指定条件位置。

习题：

  • 167.Two Sum II - Input array is sorted (Easy)
  • 633.Sum of Square Numbers (Easy)
  • 345.Reverse Vowels of a String (Easy)
  • 680.Valid Palindrome II (Easy)

在遇到有序数组时，应该优先考虑使用双指针来解决问题。因为通过使用两个指针来遍历数组可以有效减少时间复杂度和空间复杂度。
